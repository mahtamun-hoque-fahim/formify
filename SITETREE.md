## Field Reference

- **Auth levels:** `public`, `authenticated`, `admin`, `role:[name]`
- **Status progression:** `planned` → `built` (repo-maintainer, as routes get coded) → `verified` (sentinel or valley-of-death, once auth gates confirmed against actual code)
- **API routes do not belong here.** Backend endpoints stay in PLANNER.md's "API Routes" table. This file is page/UI routes only.
- **No `/pricing` route** — Formify is free during pre-launch. Pricing page added only when Fahim triggers public monetization.
